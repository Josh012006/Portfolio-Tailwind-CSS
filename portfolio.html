Ce qui suit regroupe l'ensemble de mes projets par compétence. Vous y trouverez à chaque fois un aperçu du résultat du projet et également une description brève du processus d'obtention ou de certaines particularités du projet lorsqu'il y a lieu de le faire.

HTML

Le HTML est un langage utilisé pour définir la structure de la page web. Il est donc essentiel à la création de tout site web. N'ignorant pas son importance, je me suis donc assuré que je maitrisais correctement ce langage.

CSS

Le CSS est le langage qui permet d'habiller le site web structuré grâce à du HTML. Mes connaissances dans ce domaine ne s'arrête pas seulement aux simples propriétés courantes mais s'étendent jusqu'aux media queries et même au SASS.

Bootstrap CSS

Une des choses qui importe aussi dans le développement web est l'obtention d'un site réactif qui est beau à voir sur n'importe lequel des écrans. Pour cela, on peut utiliser la framework Bootstrap dont je maîtrise le fonctionnement et l'utilisation des classes. Ici vous pouvez voir deux pages web. A chaque fois vous avez la version grand écran et aussi la version mobile qui est obtenue grâce à Bootstrap CSS en rendant le site web réactif.

Tailwind CSS

Tailwind est aussi une framework très utile et intéressante pour obtenir des sites web réactifs. J'ai fait en sorte de la maitriser ainsi que tout ce qui à rapport à la modification de ses fonctionnalités et des classes utilitaires. Vous avez ici le site web qui est présenté en version grand écran mais aussi en version mobile. Cette réactivité est obtenue grâce à Tailwind CSS.
